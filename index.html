Projects

ChordStorm

ChordStorm is a web app made for GoatHacks 2024 that recommends music based on the current weather and your listening history. The app was built using Flask for the backend, OpenAI's GPT API for the music recommendations, and Spotify's API for the user's listening history. If you'd like to see the code, you can find it on my GitHub. We're working on a way to deploy the app so that it can be used by anyone, but for now, there is a demo video on the Devpost page.

Minesweeper

I'm a long-time fan of the game Minesweeper so while I was bored over my school break, I decided to make my own version of the game. I used Python and the Pygame library to make a complete re-creation of the game, including chording. If you'd like to see the code, you can find it on my GitHub.

Neural Network

While I was home from school for the summer, I came across a YouTube video describing how to create a neural network from scratch using only Python and Numpy (no TensorFlow or SciKit-Learn). I decided to give it a try and ended up creating a neural network that could classify handwritten digits with around 85% accuracy on the MNIST dataset. If you'd like to see the code, you can find it here.
